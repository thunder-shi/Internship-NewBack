# Spring Boot 项目运行说明

## ✅ 已完成的配置

1. ✅ Maven Wrapper 已配置（无需安装 Maven）
2. ✅ Maven 镜像已配置为阿里云（加速下载）
3. ✅ Java 21 已安装

## 🚀 运行方式

### 方式一：使用批处理文件（推荐）

直接双击运行 `run.bat` 文件，它会自动：
- 设置 JAVA_HOME 为 Java 21
- 运行 Spring Boot 应用

### 方式二：使用命令行

在项目根目录打开 PowerShell 或 CMD，执行：

```bash
# 设置 JAVA_HOME（当前会话有效）
$env:JAVA_HOME='C:\Program Files\Java\jdk-21'

# 运行项目
.\mvnw.cmd spring-boot:run
```

### 方式三：使用 IDE（最简单）

1. 在 IDE（IntelliJ IDEA/Eclipse）中打开项目
2. 找到 `src/main/java/com/example/newcms/NewcmsApplication.java`
3. 右键选择 "Run" 或点击运行按钮

**注意**：如果使用 IDE，需要先设置项目的 JDK 为 Java 21：
- IntelliJ IDEA: File -> Project Structure -> Project -> SDK -> 选择 Java 21
- Eclipse: 右键项目 -> Properties -> Java Build Path -> Libraries -> 添加 Java 21

## 📝 永久设置 JAVA_HOME（可选）

如果想永久设置 JAVA_HOME 环境变量：

1. 右键"此电脑" -> 属性 -> 高级系统设置
2. 点击"环境变量"
3. 在"系统变量"中新建或编辑 `JAVA_HOME`
4. 值设置为：`C:\Program Files\Java\jdk-21`
5. 在 Path 中添加：`%JAVA_HOME%\bin`

## ✅ 验证运行

项目启动成功后：
- 控制台会显示：`Started NewcmsApplication in X.XXX seconds`
- 应用运行在：**http://localhost:8080**

## 🔧 常见问题

1. **端口被占用**：修改 `src/main/resources/application.properties` 中的 `server.port`
2. **编译失败**：确保 JAVA_HOME 指向 Java 21
3. **依赖下载慢**：已配置阿里云镜像，应该很快
